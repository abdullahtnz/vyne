Welcome to Vyne

Vyne is a high-performance, stack-based interpreted language designed for modularity and clear execution flow.

Core Principle: The VM always does the pop.

Syntax Basics

Declaring variables and functions is intuitive:

var energy = 100;
function calculate(factor) {
    return energy * factor;
}

Postfix Operators

Vyne supports postfix increment and decrement. These operations return the original value before the modification. 

var x = 5;
log(x++); // Outputs 5
log(x);   // Outputs 6

Groups

Groups allow you to categorize logic into tree-based structures, preventing global namespace pollution. 


            group Physics {
                gravity = 9.81;
                sub getWeight(mass) {
                    return mass * gravity;
                }
            };
